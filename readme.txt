In this repository we report a simple implementation of the method proposed in

Evgenii Chzhen, Mohamed Hebiri, Christophe Denis, Luca Oneto, and Massimiliano Pontil, Leveraging Labeled and Unlabeled Data for Consistent Fair Binary Classification, Neural Information Processing Systems, 2019

on the Adult Dataset (https://archive.ics.uci.edu/ml/datasets/adult) using Random Forests with no hyper-parameters tuning.
